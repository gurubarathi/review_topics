Css @ Rule

CSS statements that instruct the CSS how to behave in particular conditions

syntax:@identifier (condition) {}

@charset

It defines the character set to be used by the website. We can use it easily if the CSS file contains non-ASCII characters
sometimes degree 'o' symbols not available in ascii we use this.

@media

@font-face:

It specifies a custom font with which to display the text. Firstly, we must define the font name and then provide the URL of the font to be loaded from.

@keyframes

It gradually changes from one set of CSS to another set of CSS styles.

@import

The @import CSS at-rule is used to import style rules from other stylesheets.
